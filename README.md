Morse Keyboard respects user privacy.

## Data collection
The application does **not collect, store, or transmit** any personal or sensitive data, including typed text, passwords, messages, or contacts.

## Keyboard input
All keyboard input is processed **locally on the device** only to provide typing functionality. Typed content is not logged, recorded, or shared outside the active input field.

## Local storage
User preferences and user-added words are stored locally on the device using Android system storage (SharedPreferences). This data never leaves the device.

## Third-party services
Morse Keyboard does not use analytics services, advertising SDKs, trackers, or any third-party data processors.

## Internet access
The application works fully offline and does not send or receive data over the internet.

## User control
Users can remove all locally stored data at any time using Android system settings.
