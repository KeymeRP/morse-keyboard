# Privacy Policy — Morse Keyboard

Last updated: December 2025

## Overview
Morse Keyboard is an Android input method (keyboard) application that allows users to input text using Morse code.

We respect user privacy and are committed to protecting it.

## Data Collection
Morse Keyboard **does not collect, store, transmit, or share any personal data**.

Specifically:
- No typed text is collected or sent anywhere
- No keystrokes are logged
- No network connections are used
- No analytics, tracking, or advertising SDKs are included
- No accounts or authentication are required

## Local Processing
All input processing happens **locally on the user's device**.
User-added words and settings are stored only in local device storage using Android system preferences.

## Permissions
The app uses only standard Android permissions required for an input method (keyboard) to function.

## Third-Party Services
Morse Keyboard does **not** use any third-party services.

## Children’s Privacy
The app does not collect any data from users of any age, including children.

## Changes
If this privacy policy changes in the future, updates will be published on this page.
